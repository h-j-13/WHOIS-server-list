域名 WHOIS 服务器列表
===================
使用方法
---
详见 ```WHOIS_server_list.data```   

说明
---
收录了**绝大部分**TLD与二级域的WHOIS服务器     

基础数据来源 
* [IANA](http://www.iana.org/domains/root/db/) 

获取方法
* 通过 linux 命令            
```$ whois .cn -h whois.iana.org```         
查询WHOIS服务器      

* 通过 IANA       
[IANA提供的查询网址](https://www.iana.org/whois)

* 通过域名注册商的数据

* 通过其他或方式得到的数据
